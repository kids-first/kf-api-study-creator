Kids First Study Creator¶

About¶

The Study Creator is a GraphQL API backing the study creation process in Kids First. It manages studies, batches, and the files within, allowing users to version, validate, configure, and submit the data they are providing for inclusion in Kids First.

Quickstart¶

Getting started is easy! To start running instance of the service complete with data, install Docker and Docker Compose then run:

docker-compose up --build -d

This will build and download necessary images and run them creating the primary webserver at http://localhost:8080. An admin user will also be created by default and can log in to the Django admin panel at http://localhost:8080/admin using the default credentials:

username: admin
password: admin

The GraphQL endpoint and GraphiQL interface is at http://localhost:8080/grapql which can be used to interact with the pre-populated database of mock data.
